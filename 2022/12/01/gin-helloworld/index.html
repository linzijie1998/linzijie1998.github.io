【GIN学习笔记】Gin框架的安装和使用

发布于 2022-12-01 GIN学习笔记  golang gin 

Ⅰ Gin框架安装

  1. 初始化Go项目，使用go mod管理项目

    go mod init

  2. 下载并且安装Gin框架

    go get -u github.com/gin-gonic/gin

Ⅱ Gin示例

package main

import (
    "net/http"

    "github.com/gin-gonic/gin"
)

func main() {
    // 创建一个默认路由引擎
    router := gin.Default()
    // 使用GET方法请求/hello路径时返回JSON数据
    router.GET("/hello", func(ctx *gin.Context) {
        ctx.JSON(http.StatusOK, gin.H{
            "message": "hello, world!",
        })
    })
    // 在本地启动HTTP服务，端口号为8080
    router.Run(":8080")
}

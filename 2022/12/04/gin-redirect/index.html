【GIN学习笔记】Gin框架重定向和转发

发布于 2022-12-04 GIN学习笔记  golang gin 

重定向和转发的区别：

  • 相同点：页面都会实现跳转；
  • 不同点：请求转发的地址栏url不会改变，而重定向会变成新的url。

  1. 在Gin框架中实现重定向：

    router.GET("/index", func(ctx *gin.Context) {
    ctx.Redirect(http.StatusMovedPermanently, "https://www.baidu.com")
})

    其中http.StatusMovedPermanently表示状态码301：即永久重定向

  2. 在Gin框架中实现转发：

    router.GET("/a", func(ctx *gin.Context) {
    // 跳转到 /b 对应的路由函数
    ctx.Request.URL.Path = "/b" // 修改为b页面的URL
    router.HandleContext(ctx)   // 转发实现
})

router.GET("/b", func(ctx *gin.Context) {
    ctx.JSON(http.StatusOK, gin.H{
        "message": "page b",
    })
})

    转发的步骤：(1) 修改ctx的请求链接的路径，(2) 然后使用HandleContext()方法实现转发

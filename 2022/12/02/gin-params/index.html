【GIN学习笔记】Gin框架获取参数

发布于 2022-12-02 GIN学习笔记  golang gin 

本节主要总结如何从获取从前端传来的参数，例如querystring、form表单、json数据等等。以及如何使用ShouldBind()方法快速高效地自动获取不同类型的数据参数，本节重点为第Ⅳ小节内容。

Ⅰ 获取QUERYSTRING参数

querystring指URL中?后携带的参数，例如/search?language=golang&tag=web，获取这些参数的方法如下：

router.GET("/search", func(ctx *gin.Context) {
    language := ctx.Query("language")
    tag := ctx.Query("tag")
    
    ctx.JSON(http.StatusOK, gin.H{
        "language": language,
        "tag":      tag,
    })
})

使用Query()方法获取querystring的参数时，如果没有匹配到key则会返回空字符串。我们可以使用DefaultQuery()方法来设置默认值，避免出现空字符串：

router.GET("/search", func(ctx *gin.Context) {
    language := ctx.DefaultQuery("language", "unknown")
    tag := ctx.DefaultQuery("tag", "unknown")

    ctx.JSON(http.StatusOK, gin.H{
        "language": language,
        "tag":      tag,
    })
})

Ⅱ 获取FORM表单参数

router.POST("/user", func(ctx *gin.Context) {
    username := ctx.PostForm("username")
    password := ctx.PostForm("password")
    ctx.JSON(http.StatusOK, gin.H{
        "status":   "ok",
        "username": username,
        "password": password,
    })
})

和querystring的获取一样，form表单同样可以使用DefaultPostForm()设置默认值，username := ctx.DefaultPostForm("username", "unknown")

Ⅲ 获取URL路径

router.GET("/blog/:year/:month", func(ctx *gin.Context) {
    year := ctx.Param("year")
    month := ctx.Param("month")
    ctx.JSON(http.StatusOK, gin.H{
        "year":  year,
        "month": month,
    })
})

Ⅳ *参数绑定

为了更方便地获取请求参数，我们使用ShouldBind()方法自动识别，并且提取请求中的querystring、form表单、JSON数据等参数到结构体中。首先定义一个用于用户登录的结构体Login：

type Login struct {
    User     string `json:"user" form:"user" binding:"required"`
    Password string `json:"password" form:"password" binding:"required"`
}

  1. 绑定JSON格式数据的示例 (需要在Login结构体中利用反射机制设置字段名称，例如json:"user")：

    router.POST("/loginJSON", func(ctx *gin.Context) {
    var login Login
    err := ctx.ShouldBind(&login)
    if err == nil {
        fmt.Printf("login: %v\n", login)
        ctx.JSON(http.StatusOK, gin.H{
            "user":     login.User,
            "passowrd": login.Password,
        })
    } else {
        ctx.JSON(http.StatusBadRequest, gin.H{
            "error": err.Error(),
        })
    }
})

  2. 绑定FORM表单数据的示例 (需要在Login结构体中利用反射机制设置字段名称，例如form:"user")：

    router.POST("/loginFORM", func(ctx *gin.Context) {
    var login Login
    err := ctx.ShouldBind(&login)
    if err == nil {
        fmt.Printf("login: %v\n", login)
        ctx.JSON(http.StatusOK, gin.H{
            "user":     login.User,
            "passowrd": login.Password,
        })
    } else {
        ctx.JSON(http.StatusBadRequest, gin.H{
            "error": err.Error(),
        })
    }
})

  3. 绑定QUERYSTRING数据的示例，/loginQS?user=linzijie&password=123456：

    router.GET("/loginQS", func(ctx *gin.Context) {
    var login Login
    err := ctx.ShouldBind(&login)
    if err == nil {
        fmt.Printf("login: %v\n", login)
        ctx.JSON(http.StatusOK, gin.H{
            "user":     login.User,
            "passowrd": login.Password,
        })
    } else {
        ctx.JSON(http.StatusBadRequest, gin.H{
            "error": err.Error(),
        })
    }
})

在使用ShouldBind()方法时需要注意：

  • 传入的变量需要加上取地址符号&，类似于C++的引用，以保证能够为外面声明的变量赋值；
  • 用于保存参数的结构体，成员变量需要大写，以保证能够外部访问到；
  • 如果前端提交的JSON或者FORM表单的字段名称和声明的结构体的字段名不一样(例如大小写问题)，应当使用反射机制对结构体成员进行映射；

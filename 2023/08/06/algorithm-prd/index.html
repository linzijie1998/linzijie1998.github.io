PRD伪随机算法

发布于 2023-08-06 算法笔记  二分查找 伪随机 

1 问题描述

1：进行第一次攻击，暴击概率为C%，如果未暴击则进入2，如果暴击则进行下一次暴击计算；

2：进行第二次攻击，暴击概率为2C%，如果未暴击则进入3，如果暴击则进行下一次暴击计算；

3：进行第三次攻击，暴击概率为3C%，如果未暴击则进入4，如果暴击则进行下一次暴击计算；

4：进行第四次攻击，暴击概率为4C%，如果未暴击则进入5，如果暴击则进行下一次暴击计算；

……

n：进行第n次，暴击为nC% >= 100%，此时必定暴击。

对于每个概率值C，都可以计算出其暴击率的期望攻击次数E；

给定的概率P% (5 <= P <= 50, p = 100 / E)，求出常数C。

2 测试用例

输入：20

输出：5.57

输入：25

输出：8.475

3 解析

3.1 期望攻击次数的计算

已知初始暴击率为C，那么最差情况需要进行n = ceil(1/c)次攻击才能暴击。和古典概率模型不同的是，古典概率模型不存在保底机制，因为它每次攻击都是一次独立的随机事件，而这种伪随机模型则给了保底暴击的累计攻击次数。

现在我们需要计算在n次的攻击次数内暴击的概率分布情况，这里假设初始暴击率为25%，即C = 0.25，保底攻击次数则为n = ceil(1 / 0.25) = 4，下面我们对四次攻击进行模拟：

  • 第一次攻击：暴击率为1×0.25=0.25，第一次就暴击的概率也为0.25；
  • 第二次攻击：暴击率为2×0.25=0.5，第二次就爆击的概率应该是0.5×(1-0.25)=0.375；
  • 第三次攻击：暴击率为3×0.25=0.75，第三次就爆击的概率应该是0.75×(1-0.25-0.375)=0.28125；
  • 第四次攻击：暴击率为4×0.25=1.0，第四次才暴击的概率应该是1.0×(1-0.25-0.375-0.28125)=0.09375。

值得注意的是：每次攻击的暴击率不等于暴击落在该次攻击的概率；

得到暴击落在每次攻击的概率后我们就可以计算得到该伪随机模型的攻击次数期望：E = 1×0.25+2×0.375+3×0.28125+4×0.09375=2.21875 ，即平均攻击2.21875次会暴击一次。

3.2 暴击率的拟合方法

已知期望攻击次数E=2.21875我们可以得到暴击率为P = 1÷E=1÷2.21875=0.4507042253521127，而我们的目的是尽可能让P去拟合在给定的暴击率。根据题目描述我们可以知道初始暴击率C一定小于给定的暴击率P，那么我们可以使用二分查找的方法来找到这个最接近P的值C：

  • 设定两个指针left和right分别指向0和P，计算mid=(left + right) / 2的暴击率estimateP；
  • 如果estimateP大于P，则right指向mid；如果estimateP小于P，则left指向mid；
  • 如果estimateP等于P（或在某个阈值范围内），则表示拟合成功，当前mid值即为所求的初始暴击率C；

4 代码

func estimateC(c float64) float64 {
    nMax := math.Ceil(1 / c) // 最多攻击次数
    curCRIT := 0.            // 第i次的为暴击的概率
    sumCRIT := 0.            // 累计的暴击概率，最后一次攻击过后累加到1
    expected := 0.           // 攻击次数期望
    // 计算攻击次数期望：攻击次数乘以暴击概率
    for i := 1.; i <= nMax; i++ {
        curCRIT = math.Min(i*c, 1.) * (1 - sumCRIT)
        sumCRIT += curCRIT
        expected += i * curCRIT
    }
    // 估算暴击率
    return 1. / expected
}

func BinarySearchC(p, threshold float64) float64 {
    left, mid, right := 0., 0., p
    for {
        mid = (left + right) / 2.
        estimateP := estimateC(mid)
        if math.Abs(estimateP-p) < threshold {
            break
        }
        if estimateP > p {
            right = mid
        } else {
            left = mid
        }
    }
    return mid
}
